
# AGENTS.md に関して

##  VSCode 上のGitHub Copilot（Agent Mode）で実行されることを想定

- **Blackbird** による構文木ベースのコンテキスト把握が有効であること
    - このため、追加のコンテキスト抽出指示は行わない
- ローカル PC または codespcaeで実行している前提
- ターミナル実行権限がある前提
- Git コマンド実行が許可されている前提

## そのほかのAIコーディングテクノロジーを用いる場合

- Kiri MCPや、Serena MCPなどコンテクストを高精度に探索できるツールの利用を強く推奨します
