# Documentation Update Workflow

Used when code changes require updates to README/USAGE.

## Draft Flow
- Create draft under `docs/drafts/`:
  `README.update-<task-slug>.md`
- Keep changes tied to feature branch.

## Merge Flow
- On merging to main:
  - Apply draft to actual README/USAGE.
  - Remove the draft file.

## Rules
- Ensure examples are updated.
- Maintain consistent style.
